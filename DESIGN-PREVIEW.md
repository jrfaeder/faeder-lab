- Font: Inter (modern, clean)
- Weight: Bold (700)
- Size: 2.5rem (h1) down to 1.25rem (h4)

**Body Text:**
- Font: Inter
- Weight: Regular (400)
- Size: 16px base
- Line height: 1.6 (comfortable reading)

### Spacing

- Section padding: 4-6rem
- Card gaps: 2rem
- Content margins: 1-2rem
- Mobile padding: 1rem

### Interactive Elements

**Buttons:**
- Rounded corners (6px)
- Subtle hover effect (slight lift + shadow)
- Transition animations (0.2s)

**Cards:**
- Subtle shadow
- Hover effect: Lift up + enhanced shadow
- Rounded corners (8px)

**Navigation:**
- Sticky header (stays at top when scrolling)
- Smooth underline on hover
- Active page highlighted
- Mobile: Hamburger menu with slide-down

### Responsive Behavior

**Desktop (> 1024px):**
- 3-column grid for highlight cards
- 2-column hero (text + photo)
- 3-4 team members per row

**Tablet (640-1024px):**
- 2-column grids
- Maintained spacing
- Readable text sizes

**Mobile (< 640px):**
- Single column layout
- Stacked navigation
- Full-width cards
- Larger touch targets

## Special Features

1. **Smooth Scrolling** - Page navigation feels fluid
2. **Lazy Loading** - Images load as you scroll
3. **Dark Mode Ready** - Footer demonstrates dark theming
4. **Accessibility** - ARIA labels, semantic HTML
5. **SEO Optimized** - Meta tags, structured data
6. **Fast Loading** - Static HTML, minimal JavaScript
